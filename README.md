# TravelSuperApp（前端原型）

当前仓库先实现 **Android（Jetpack Compose）前端页面骨架**：主页 superapp 布局、入口页占位、底部导航（含中间“问AI”按钮）。

## 开发环境

- Android Studio（推荐最新版稳定版）
- JDK：使用 Android Studio 自带的 Embedded JDK 即可

## 如何运行

1. 用 Android Studio 打开项目根目录
2. 等待 Gradle Sync 完成
3. 连接手机或启动模拟器，点 Run

> 注意：`local.properties` 不会提交到仓库，每个人机器会自动生成自己的 SDK 路径配置。
