## 1. 项目概述

### 1.1 核心功能

| 功能模块 | 描述 |
|----------|------|
| **找导游** | 核心功能，支持按标签、地区、距离、评分筛选导游 |
| **AI 助手** | DeepSeek 驱动的旅行规划 AI |
| **旅行模块** | 结伴骑游、乘车出游、组团出游（含路线推荐） |
| **点亮地图** | 用户在地图上添加/查看旅行标记 |
| **路线跟踪** | GPS 记录骑行/步行轨迹 |
| **美食/购物** | 基于 GPS 显示周边商家 |
| **发布/社区** | 用户发布游记、浏览动态 |

### 1.2 项目信息

| 项目 | 信息 |
|------|------|
| 项目名称 | Travel Superapp |
| 平台 | Android (Kotlin) |
| 最低 SDK | 24 (Android 7.0) |
| 目标 SDK | 34 (Android 14) |
| UI 框架 | Jetpack Compose |
| 后端服务 | Supabase |
| AI 服务 | DeepSeek API |

## 10. API 接口文档

### 10.1 Supabase REST API

**基础 URL**: `https://wqbjmbktknonxqtmhsrk.supabase.co/rest/v1`

**请求头**:
```
apikey: <ANON_KEY>
Authorization: Bearer <JWT>
Content-Type: application/json
Prefer: return=representation
```

### 10.2 核心接口

#### 获取导游列表
```
GET /guides?select=*
```
Query 参数:
| 参数 | 说明 | 示例 |
|------|------|------|
| province | 省份 | `province=eq.陕西` |
| city | 城市 | `city=eq.西安` |
| tags | 标签 | `tags=cs.{"骑行专家"}` |
| rating | 评分 | `rating=gte.4.5` |
| order | 排序 | `rating.desc` |

#### 获取景区列表
```
GET /attractions?select=*
```
Query 参数:
| 参数 | 说明 | 示例 |
|------|------|------|
| city | 城市 | `city=eq.西安` |
| type | 类型 | `type=eq.历史古迹` |

#### 创建导游订单
```
POST /guide_orders
Body: {
  "guide_id": "uuid",
  "user_id": "uuid",
  "travel_date": "2024-04-01",
  "days": 3,
  "status": "pending"
}
```

#### 获取评价列表
```
GET /guide_reviews?select=*&guide_id=eq.<guide_id>
```

#### 创建游记
```
POST /posts
Body: {
  "user_id": "uuid",
  "title": "标题",
  "content": "内容",
  "images": ["url1", "url2"]
}
```

### 10.3 DeepSeek API

**端点**: `POST https://api.deepseek.com/chat/completions`

**请求体**:
```json
{
  "model": "deepseek-chat",
  "messages": [
    {"role": "system", "content": "你是一个旅行助手"},
    {"role": "user", "content": "我想去西安旅游3天"}
  ],
  "stream": false
}
```
